애니메이션이 적용된 목록

부제목 위치

애니메이션 효과가 적용된 항목:

  • 'h' 를 누르면 코드 중 중요 부분을 강조합니다.
  • 'p' 를 누르면 발표자 노트를 표시합니다. (슬라이드에 노트가 있는 경우에만)
  • 'f' 를 누르면 전체화면 모드로 전환합니다.
  • 'w' 를 누르면 와이드스크린 모드로 전환합니다.
  • 'o' 를 누르면 전체 슬라이드 보기 모드로 전환합니다.
  • 'ESC' 를 누르면 위의 기능을 취소합니다.

  }
